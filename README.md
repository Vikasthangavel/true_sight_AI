## в—Ҳ What Is TrueSight?

> *"Seeing through the synthetic."*

TrueSight AI is a forensic video analysis platform that detects AI-generated deepfakes using multi-frame inference. Upload a video or drop in a YouTube link вҖ” TrueSight extracts strategic frames, runs each through a trained Roboflow model, and delivers a verdict backed by confidence scores, risk classification, and a downloadable PDF forensic report.

Built for a **Cyber Hackathon** targeting the growing threat of AI-generated media in fraud, identity attacks, and disinformation campaigns.

## в—Ҳ Getting Started

### 1 вҖ” Clone

```bash
git clone https://github.com/Vikasthangavel/true_sight_AI.git
cd true_sight_AI
```

### 2 вҖ” Create a virtual environment

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3 вҖ” Install dependencies

```bash
pip install -r requirements.txt
```

### 4 вҖ” Set your API key

Create a `.env` file in the project root:

```env
ROBOFLOW_API_KEY=your_roboflow_api_key_here
```

> вҡ пёҸ Never commit your `.env` file вҖ” it is already listed in `.gitignore`.
>
> Get a free API key at [roboflow.com](https://roboflow.com)

### 5 вҖ” Run

```bash
python app.py
```

App starts at **`http://localhost:5000`**
